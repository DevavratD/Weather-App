WeatherX is a sleek and interactive web application that provides weather forecasts based on your location. It fetches real-time weather data and displays current conditions, hourly forecasts, and daily trends, helping you stay prepared for the day ahead!

---

## Features

- **Current Weather**: Displays real-time weather information such as temperature, humidity, and weather conditions.
- **Hourly Forecast**: Get a 6-hour forecast for the day, including temperature and weather icons.
- **Daily Forecast**: View high and low temperatures for the 3 days.
- **Unit Conversion**: Toggle between units to suit your preference.
- **User-friendly Interface**: Clean and intuitive UI for easy navigation.

---

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript (Vanilla JS)
- **Backend**: Node.js , Express.js
- **Weather API**: https://www.weatherapi.com/
- **Deployment**: Render (for cloud hosting)

## Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) and [npm](https://npmjs.com/) installed on your system.

### 1. Clone the repository

```bash
git clone [https://github.com/DevavratD/weatherx.git](https://github.com/DevavratD/weatherx.git)
```
### 2. Install dependencies

```bash
cd weatherx
npm install
```

### 3. Setup your api key in code

### 4. Start the server
```bash
npm start
```
Your app should now be running on http://localhost:3000.

## Deployment
This project is deployed on Render, which automatically handles the hosting of your app with continuous deployment.

To deploy it yourself, follow the Render documentation to connect your GitHub repository to Render:
1. Push your code to GitHub.
2. Create a new project on Render.
3. Connect your GitHub repository to Render.
4. Render will automatically detect the app, install dependencies, and start your server.
